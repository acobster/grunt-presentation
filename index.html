Automating your development workflow with Grunt

Coby Tamayo

What is Grunt?

  • A general-purpose task runner for devs
  • Built on Node.js

What sort of tasks?

  • Well, anything...that can be automated
  • Plugin architecture:
    • Download generic tasks with npm
    • Register tasks in Gruntfile.js
    • Configure tasks with initConfig()

How does it work?

  • Core tasks are Node.js plugins
  • Declare dependencies in package.json
  • Install these plugins with npm
  • Run them with grunt
